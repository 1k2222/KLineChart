# dispose(dcs)
`dispose` 用于销毁一个图表。

## 参考 {#reference}
<!-- @include: @/@views/api/references/chart/dispose.md -->

### 参数
- `dcs` 图表实例，`dom` 元素或者 `dom` 元素id。

### 返回值
`dispose` 返回 `undefined` 。
